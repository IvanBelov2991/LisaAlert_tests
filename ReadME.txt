Архитектура проекта:

features
    steps - здесь хранятся степы behave
    tests_main_page - здесь хранятся тесты
    environment.py - настройки фикстуры для запуска
locators - здесь хранятся локаторы (общие и отдельные по страницам/проектам)
pages
    base_page.py - здесь хранятся общие функции селениум
    main_page.py - здесь хранятся функции селениум, кастомизированные под определенный проект
data.py - здесь хранятся переменные по проекту (url и словари)
requirements.txt -здесь хранятся библиотеки
